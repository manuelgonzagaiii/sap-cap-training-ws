# Overview and roadmap

Package for production (CF or Kyma), bind services, and plan a safe schema migration.

## What this lesson covers

- cds add hana,xsuaa,mta / kyma / approuter
- cds build and cds up (CF default / Kyma)
- service bindings (cds bind)
- schema evolution (cds deploy --script)
- GitHub Actions facet
